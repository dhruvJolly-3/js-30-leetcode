# Day 02 — Counter

**LeetCode Link:** https://leetcode.com/problems/counter/

**Difficulty:** Easy

**Topic:** Closures

---

## Problem Statement

> Given an integer `n`, return a counter function. This counter function initially returns `n` and then returns 1 more than the previous value every subsequent time it is called.

## Examples

```
Input:  n = 10, calls = ["call","call","call"]
Output: [10, 11, 12]

Input:  n = -2, calls = ["call","call","call","call","call"]
Output: [-2, -1, 0, 1, 2]
```

## Constraints

- -1000 <= n <= 1000
