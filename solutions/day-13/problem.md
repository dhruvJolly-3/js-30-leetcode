# Day 13 — Sleep

**LeetCode Link:** https://leetcode.com/problems/sleep/

**Difficulty:** Easy

**Topic:** Promises / Async / setTimeout

---

## Problem Statement

> Given a positive integer `millis`, write an async function that sleeps for `millis` milliseconds. It can resolve any value.

## Examples

```
Input:  millis = 100
Output: 100 (resolves after 100ms)

Input:  millis = 200
Output: 200 (resolves after 200ms)
```

## Constraints

- 1 <= millis <= 1000
