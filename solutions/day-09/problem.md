# Day 09 — Return Length of Arguments Passed

**LeetCode Link:** https://leetcode.com/problems/return-length-of-arguments-passed/

**Difficulty:** Easy

**Topic:** Rest Operator

---

## Problem Statement

> Write a function `argumentsLength` that returns the count of arguments passed to it.

## Examples

```
Input:  args = [5]
Output: 1

Input:  args = [{}, null, "3"]
Output: 3
```

## Constraints

- 0 <= args.length <= 100
